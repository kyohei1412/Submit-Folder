# Smart Lock Key（ESP32 スマートロック）

## 概要
ESP32とサーボモータ（MG90S）を使用して、Webブラウザから施錠・解錠できるスマートロックを作成しました。  
ESP32がWi-Fiアクセスポイント（AP）として動作し、スマホから接続して操作できるようにしています。

## 主な機能
- ESP32をWi-Fiアクセスポイントとして起動（SSID/パスワードで接続）
- ブラウザ画面からボタン操作でサーボ角度を変更
- 施錠 / 解錠の状態を画面に表示
- 現在角度（サーボの角度）を表示

## 使用部品・技術
### ハードウェア
- ESP32
- サーボモータ：MG90S
- 外部電源（5V）
- ジャンパ線

### ソフトウェア
- Arduino IDE / PlatformIO
- ESP32 WiFi（WiFi.h）
- WebServer（WebServer.h）
- ESP32Servo（ESP32Servo.h）

## 配線例
- サーボ信号線：GPIO14（例）
- サーボ電源：5V（外部電源）
- サーボGND：GND（ESP32と外部電源のGNDは共通）

※外部電源を使用する場合は、ESP32のGNDと外部電源のGNDを必ず共通にしています。

## 動作手順
1. ESP32にプログラムを書き込みます
2. スマホで Wi-Fi一覧から ESP32のSSID（例：ESP32_SERVO）に接続します
3. ブラウザで指定のIPアドレスにアクセスします（例：192.168.4.1）
4. 表示されたボタンで施錠 / 解錠を操作できます

## 工夫した点
- ESP32をAPモードで動作させ、ルーターが無い環境でも操作できるようにしました
- ブラウザUIをシンプルにして、スマホでも押しやすいボタンサイズにしました
- サーボの電源不足による動作不安定を避けるため、外部5V電源を使用しました
